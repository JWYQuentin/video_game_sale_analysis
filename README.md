# Video Game Sales Analysis

Analyzing 16,000+ video games to predict commercial hits and uncover regional market segments.

## Project Structure
- `notebooks/01_data_cleaning.ipynb` — Data cleaning & preprocessing
- `notebooks/03_eda.ipynb` — Exploratory data analysis
- `notebooks/04_feature_engineering.ipynb` — Feature engineering (16 new features)
- `notebooks/05_classification.ipynb` — Hit prediction modeling
